## üöÄ Features and How to Use

The extension supports:

1.  **Bulk Download (via Popup):**
    * Go to your `Library` or a song list on Suno.ai.
    * **Be sure to scroll to the bottom of the page** to load all songs into the DOM.
    * Click on the extension icon and select **"Download All WAVs"** or **"Download All MP3s"**. You can configure the delay between downloads.


### ‚öôÔ∏è Download Settings

The following options are available in the popup window:
* **Delay (seconds):** The delay between consecutive downloads (to reduce the load on Suno's servers).
* **Create Subfolder:** Creates a `SUNO_Workspace_Name/` folder to organize files.
* **Include Cover Art:** Downloads the corresponding JPEG cover art along with the audio file.
* **Include UUID in Filename:** Adds a unique identifier (`UUID`) to the filename to avoid conflicts.

## üõ†Ô∏è Installation (Developer Mode)

Since this extension is not published in the Chrome Web Store, it must be installed in developer mode.

1.  **Download the files:** Download all extension files (for example, as a ZIP archive) and **unzip** them into a new empty folder (`suno-downloader`).
2.  **Open the extensions page:** In Google Chrome, go to `chrome://extensions/`.
3.  **Enable Developer Mode:** In the upper-right corner of the page, find and activate the **"Developer mode"** switch.
4.  **Load the extension:** Click the **"Load unpacked"** button and select the folder you created in step 1.

After this, the extension's icon will appear on the Chrome toolbar.
